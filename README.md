# 🎬 Movie Finder App

A Python desktop application to search and explore movies using the TMDB API.

## Features
- Search movies by title
- View movie posters, ratings, and release dates
- Click on a movie to see full details

## Tech Stack
- Python 3
- Tkinter (GUI)
- TMDB API
- Pillow (image handling)

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Get a free API key from [themoviedb.org](https://www.themoviedb.org/)
4. Create a `.env` file and add: `TMDB_API_KEY=your_key_here`
5. Run: `python main.py`
