# 🎬 Movie Finder App

A Python desktop application to search and explore movies using the TMDB API.

## 📸 Features
- 🔍 Search movies by title
- 🖼️ View movie posters
- ⭐ Ratings, release year, and overview
- 🪟 Click any movie for full details (genres, budget, runtime, tagline)
- ⌨️ Press Enter or click Search button
- 🖱️ Scroll through results
- ❌ Error handling for missing API key and network issues

## 🛠️ Tech Stack
- Python 3
- Tkinter (GUI)
- TMDB API
- Pillow (image handling)
- python-dotenv

## ⚙️ Setup
1. Clone the repo
```bash
   git clone https://github.com/YOUR_USERNAME/movie-finder.git
   cd movie-finder
```
2. Install dependencies
```bash
   pip install -r requirements.txt
```
3. Get a free API key from [themoviedb.org](https://www.themoviedb.org/)
4. Create `.env` file
